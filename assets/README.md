# Assets

## Optional hero image

The app looks great with no image at all (it uses a gradient hero by default).

If you'd like a background photo behind the headline, drop a file named
**`hero.jpg`** into this folder. The app will automatically detect it, dim it,
and fade it behind the hero text — no code changes needed.

Good choices: an abstract gradient, a soft crowd/social scene, or a calm
texture. Landscape orientation, at least 1600px wide, works best.
